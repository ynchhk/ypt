# StudyFlow 📚

A study tracker web app inspired by YPT.

## Features
- ⏱ Study timer with start/pause/stop
- ✅ Todo list with task management
- 📊 Today's stats (sessions, avg length)
- 📈 Weekly activity chart

## Tech Stack
HTML · CSS · JavaScript · Chart.js · Lucide Icons
